# 🛍️ DigiTools

## Overview
DigiTools is a digital tools buying website where users can browse products, view detailed information, add products to the cart, remove selected products, and proceed to checkout. The project focuses on clean UI, responsive design, and interactive cart functionality.

## Live Link
https://digitools-by-nur.netlify.app/

## Technologies Used
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON

## Main Features
- Product and cart section toggle
- Add to cart, remove item, and proceed to checkout
- Toast notifications for cart actions
- Dynamic cart count in navbar
- Responsive card-based product layout

## Dependencies
- react
- react-dom
- react-toastify
- react-icons
- tailwindcss
- daisyui
